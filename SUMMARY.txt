                          QUICK START (3 STEPS)
================================================================================

Step 1: Rebuild Docker
  $ docker-compose down
  $ docker-compose up --build

Step 2: Configure Scriberr
  - Go to Settings → Scriberr
  - Leave "Scriberr Host" empty (for Docker proxy)
  - Leave "Protocol" as http

Step 3: Test Connection
  - Click "Test Connection"
  - Expected: "Scriberr proxy is healthy" ✅
